<h1 align="center">💰 被动收入工具包</h1>
<h3 align="center">7个 Claude Code Skills · 零成本自动化 · 被动收入流水线</h3>

一套用 AI + 自动化搭建被动收入系统的 Skill 合集。从建站到内容到推广，全链路覆盖。

---

## 🛠 7 个 Skills

| Skill | 一句话 | 产出 |
|-------|--------|------|
| 📰 **content-site** | 自动生成内容站，GitHub Actions 每日更新 | SEO 流量站 |
| 📱 **douyin-content** | 每日自动生成抖音图文文案 | 抖音流量 |
| 🎓 **gaokao-tool** | 高考志愿填报工具（冲稳保推荐） | 高流量工具站 |
| 🧭 **holland-test** | 霍兰德职业兴趣测试（30/60题） | 病毒传播工具 |
| 🎬 **life-sim** | 人生副本模拟器（互动视频风） | 抖音爆款内容 |
| 🔄 **social-pipeline** | 内容→社交媒体自动转换流水线 | 多平台分发 |
| 🛠 **toolbox** | 在线工具箱（15+实用工具） | 工具站流量 |
