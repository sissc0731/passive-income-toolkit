# 霍兰德职业兴趣测试

## 用途
搭建一个 RIASEC 霍兰德职业兴趣测试网站。双版本，病毒传播，高考志愿填报流量入口。

## 特点

| 版本 | 题数 | 用时 | 适用 |
|------|------|------|------|
| ⚡ 简洁版 | 30题 | 3分钟 | 快速了解，社交传播 |
| 📋 专业版 | 60题 | 6分钟 | 深度分析，职业规划 |

## 使用方式

```
"用 holland-test skill 帮我做一个霍兰德职业兴趣测试网站"
```

## 6种人格类型

| 代码 | 类型 | 适合专业 |
|------|------|---------|
| R | 现实型 | 机械/土木/农学 |
| I | 研究型 | 数学/物理/医学 |
| A | 艺术型 | 设计/音乐/建筑 |
| S | 社会型 | 教育/心理/护理 |
| E | 企业型 | 工商/金融/法学 |
| C | 常规型 | 会计/统计/行政 |

## 病毒传播设计

1. 结果页展示 3 字母代码 → 用户自发分享
2. 匹配专业列表 → 实用价值
3. 性格描述走心 → 共鸣引发传播
4. 分享按钮 → 一键分享朋友圈
