# 抖音图文生成器

## 用途
把内容站的文章自动转成抖音图文格式，每天一份，复制粘贴即可发布。

## 使用方式

```
"用 douyin-content skill 帮我生成今天的抖音文案"
```

## 输出格式
```
🔥 主标题（hook吸引点击）
日期

主内容（3-5句话，每句单独成行，短句为主）

━━━━━━━━━━
更多推荐：
▫️ 相关话题1
▫️ 相关话题2
━━━━━━━━━━
💬 引导收藏/互动
🔗 引导看主页

#相关标签 #标签2 #标签3
```

## 关键技巧

1. **标题要有钩子**：用数字、对比、疑问句
   - ❌ "一些时间管理技巧"
   - ✅ "3个小习惯让我每天多出2小时"

2. **正文短句**：每句话不超过20字，单独成行

3. **标签精准**：3-6个标签，不要堆砌

4. **配图建议**：每条内容配推荐图片类型

## 集成方式

在 `generate.js` 中增加 douyin 输出：
```javascript
const dyPost = `${hook} ${main.t}
${today}

${main.d}
...
#标签`;
fs.writeFileSync('douyin/latest.txt', dyPost);
```

## 发布流程
1. 打开 `douyin/latest.txt`
2. 复制内容
3. 抖音 → 发布图文 → 粘贴 → 配图 → 发布
4. 全程1分钟
