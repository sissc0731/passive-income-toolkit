# 社交媒体自动化流水线

## 用途
一篇文章 → 自动生成 3 个版本：
1. 📰 网站版（详细文章 + SEO）
2. 📕 小红书版（emoji + 短句 + 标签）
3. 📱 抖音版（hook + 干货 + 引导）

## 架构

```
generate.js（每日运行）
    ├── → posts/xxx.html     (网站版)
    ├── → xiaohongshu/latest.md  (小红书版)
    └── → douyin/latest.txt      (抖音版)
```

## 使用方式

```
"用 social-pipeline skill 帮我把现有内容站接入社交媒体分发"
```

## 各平台格式

### 小红书格式
```markdown
🔥 主标题 | 日期

要点1 💡
要点2 🎯
要点3 ✨

#标签1 #标签2 #标签3
```

### 抖音图文格式
```
🎯 hook标题
日期
正文（短句分行）
#标签
```

### 转换规则
| 元素 | 网站版 | 小红书 | 抖音 |
|------|--------|--------|------|
| 标题 | H2 | emoji+短标题 | hook+数字 |
| 内容 | 段落文 | 短句+emoji | 更短句 |
| 标签 | 内部分类 | 3-5个搜索标签 | 3-5个热门标签 |
| 长度 | 500-800字 | 200-400字 | 100-200字 |

## 发布流程
1. 每天 Actions 自动运行
2. 打开对应站点的 `xiaohongshu/latest.md` 或 `douyin/latest.txt`
3. 复制 → 粘贴 → 发布
4. 全程每个平台 1 分钟
