# 人生副本模拟器

## 用途
搭建一个电影画风的人生模拟互动网页。用户选择副本（不结婚/创业/古代/出国），体验不同人生路径。

## 抖音爆款公式

这类视频在抖音爆火的核心：
1. 黑底白字电影画风
2. 年龄节点推进（25岁→30岁→35岁...）
3. 情感共鸣文案
4. 钢琴BGM
5. 竖屏9:16

## 使用方式

```
"用 life-sim skill 帮我做一个 [主题] 的人生副本"
```

### 示例
- "做一个一辈子不结婚的人生副本"
- "做一个裸辞创业的人生副本"
- "做一个回到古代的人生副本"

## 技术要点

### 视觉风格
```css
body { background: #0a0a0a; color: #d4c5b2; }
.age { font-size: 5rem; opacity: 0.15; } /* 年龄水印 */
.line { line-height: 2.2; letter-spacing: 1px; } /* 文字节奏 */
```

### 数据结构
```javascript
var scenes = [
  {age: '25岁', lines: ['第一行文字', '第二行文字']},
  {age: '30岁', lines: ['...']},
  // ...
];
```

### 交互设计
- 点击屏幕翻页（模拟视频播放感）
- 底部进度圆点
- 自动淡入动画

## 录屏变视频

用手机录屏功能录制网页播放过程 → 直接发布抖音。
